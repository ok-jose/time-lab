# 时间实验室 - 物品管理微信小程序

一个用于记录物品开启时间、限制使用日期、过期时间的智能管理工具，支持OCR识别、扫码、拍照等多种录入方式。

## 🚀 功能特性

### 核心功能
- 📅 **时间管理**：记录物品开启时间、限制使用天数、过期时间
- 🔍 **OCR识别**：拍照识别产品标签，自动提取日期信息
- 📱 **扫码录入**：扫描条形码/二维码快速添加物品
- 📸 **拍照记录**：拍摄物品照片并手动填写信息
- 📊 **时间线可视化**：直观展示物品生命周期进度
- ⏰ **智能提醒**：根据过期状态自动分类和提醒

### 用户体验
- 🎨 **实验室美学设计**：独特的工业实验室视觉风格
- 📱 **响应式界面**：适配不同屏幕尺寸
- 🔄 **本地存储**：数据安全保存，支持导入导出
- 🌙 **暗色模式**：自动适配系统外观偏好
- 🎯 **状态分类**：安全期内/即将过期/已过期三色区分

## 📁 项目结构

```
miniprogram/
├── app.js              # 应用入口，全局数据管理
├── app.json           # 应用配置，页面路由
├── app.wxss           # 全局样式，设计系统
├── project.config.json # 项目配置
├── sitemap.json       # 站点地图
├── utils/
│   └── util.js        # 工具函数（日期计算、状态判断等）
├── pages/
│   ├── index/         # 首页：物品概览
│   ├── add/           # 添加页：OCR/扫码/拍照/手动
│   ├── detail/        # 详情页：时间线+详细信息

```

## 🛠️ 技术栈

- **微信小程序原生框架**
- **WXML + WXSS + JavaScript**
- **本地存储**：`wx.setStorageSync` / `wx.getStorageSync`
- **设备API**：相机、扫码、相册、剪贴板
- **设计系统**：CSS变量、OKLCH色彩空间、4pt网格

## 📦 安装与运行

### 开发环境
1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入本项目文件夹
3. 在`project.config.json`中填入你的AppID（或使用测试号）
4. 点击"编译"运行项目

### 真机预览
1. 点击开发者工具中的"预览"按钮
2. 使用微信扫描二维码
3. 在手机上测试完整功能

## 📝 使用指南

### 1. 添加物品
- **OCR识别**：对准产品标签拍照，自动提取日期
- **扫码录入**：扫描商品条形码/二维码
- **拍照记录**：拍摄物品照片并填写信息
- **手动输入**：直接填写所有字段

### 2. 管理物品
- **首页**：按状态筛选（安全/警告/危险）
- **详情页**：查看时间线、剩余天数、备注
- **长按删除**：长按物品卡片可删除
- **下拉刷新**：更新数据状态

### 3. 设置与数据
- **提醒设置**：开启/关闭过期提醒
- **数据管理**：导入导出JSON数据
- **清空数据**：一键清除所有记录

## 🔧 开发说明

### 设计系统
- **色彩**：OKLCH色彩空间，实验室安全色系
- **字体**：系统默认字体 + 等宽字体用于数据
- **间距**：4pt网格系统（4px, 8px, 12px, 16px, ...）
- **圆角**：6px, 12px, 20px, 28px 四级圆角

### 状态计算
```javascript
// 剩余天数 = 过期日期 - 当前日期
// 状态判断：
// - 已过期：剩余天数 < 0
// - 即将过期：0 ≤ 剩余天数 ≤ 7
// - 安全期内：剩余天数 > 7
```

### 数据持久化
- 使用`wx.setStorageSync`本地存储
- 自动计算物品状态并更新
- 支持导入导出JSON格式

## 🎨 设计理念

### 实验室美学
- **精确性**：像实验室记录一样精确管理时间
- **可视化**：时间线图表直观展示生命周期
- **状态编码**：安全色系（绿/黄/红）快速识别状态
- **工业感**：冷灰色调 + 金属质感边框

### 交互细节
- **微动画**：卡片悬停、按钮点击反馈
- **手势操作**：滑动删除、长按编辑
- **状态反馈**：操作成功/失败视觉提示
- **空状态**：引导用户添加第一个物品
